# 异常服务创建

## 1. 基线说明

新建 Windows 服务是常见软件安装行为，也可能是攻击者持久化、权限维持、横向移动的方式。

## 2. 关联对象

- [[Windows Service]]
- [[HKLM_SYSTEM_CurrentControlSet_Services]]

## 3. 高风险条件

```text
服务 ImagePath 指向用户目录、Temp、Downloads、Public
服务程序无签名
服务名伪装系统服务
由脚本解释器创建服务
服务创建后立即启动并外联
服务启动类型为 Auto
```

## 4. 误报条件

```text
正规软件安装
驱动安装
企业终端管理软件部署
安全软件安装或升级
```

## 5. 证据需求

```text
服务创建者进程
完整 ImagePath
服务程序签名
创建时间
安装包来源
服务启动后的子进程和网络行为
```

# 远控软件服务常驻

## 1. 基线说明

远控软件服务常驻可能是合法 IT 运维，也可能是攻击者建立远程访问通道。不能只凭软件名判断，需要结合资产授权和行为链。

## 2. 关联对象

- [[AnyDesk]]
- [[AnyDesk Service]]
- [[services.exe -> AnyDesk.exe]]
- [[TeamViewer]]
- [[RustDesk]]
- [[ToDesk]]
- [[向日葵远控]]
- [[Chrome Remote Desktop]]

## 3. 合法条件

```text
资产台账中登记
IT 运维授权
安装路径和签名可信
用户或管理员确认
连接目标符合企业远控策略
```

## 4. 异常条件

```text
未授权终端出现远控服务
短时间内安装并外联
从临时目录运行
伴随异常登录、凭据访问、数据打包、横向移动
```

## 5. 推荐处置

优先核验授权状态和安装时间线，再关联网络、账户、文件、命令执行行为。

# 更新器外联行为

## 1. 基线说明

更新器外联本身常见，但更新器也是攻击者喜欢伪装或滥用的对象。安全分析应重点关注路径、签名、参数、目标域名和落地文件。

## 2. 关联对象

- [[GoogleUpdate.exe]]
- [[gupdate]]
- [[gupdatem]]
- [[Google Update Network]]

## 3. 正常条件

```text
路径可信
签名可信
服务名匹配
参数符合预期
目标域名符合厂商更新服务
下载文件签名可信
```

## 4. 异常条件

```text
更新器进程位于异常目录
签名异常
访问非预期域名
下载未签名可执行文件
更新后创建服务/计划任务/Run Key
```

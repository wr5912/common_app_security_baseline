# 企业安全代理服务常驻

## 1. 基线说明

企业安全代理常驻通常是合规要求；异常点既包括未知安全代理出现，也包括应安装代理缺失或被停止。

## 2. 关联对象

- [[CrowdStrike Falcon]]
- [[SentinelOne]]
- [[Wazuh Agent]]
- [[Splunk Universal Forwarder]]
- [[Elastic Agent]]

## 3. 合法条件

```text
纳入企业安全平台管理
服务路径和签名可信
管理端地址符合企业配置
资产分组与策略一致
```

## 4. 异常条件

```text
应安装终端缺失代理
安全代理被停用、卸载或替换
伪装安全代理的同名服务
代理连接未知管理端
```

## 5. 推荐处置

优先核验资产角色、授权状态、服务 ImagePath、签名、启动账户、监听端口、网络目的地址和变更时间线。不要只凭服务名判断恶意。

## 6. 证据与来源

- 来源页面：[[Windows常见应用服务基线清单]]
- 待验证：企业资产角色、授权清单、真实命令行和网络观测。

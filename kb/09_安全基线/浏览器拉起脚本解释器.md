# 浏览器拉起脚本解释器

## 1. 基线说明

浏览器拉起 PowerShell、cmd、wscript、mshta 等脚本解释器通常需要高关注。它可能来自用户操作、协议处理器、恶意扩展、下载后执行或漏洞利用。

## 2. 关联关系

- [[chrome.exe -> powershell.exe]]
- [[chrome.exe -> cmd.exe]]

## 3. 高风险条件

```text
命令行混淆
隐藏窗口
下载远程脚本
落地可执行文件
随后发生外联
写入持久化位置
```

## 4. 误报条件

```text
开发者本地测试
企业管理网页触发本地协议
用户明确手动执行下载脚本
```

## 5. 推荐处置

结合用户交互、URL、下载文件、命令行、外联、落地文件综合判断。

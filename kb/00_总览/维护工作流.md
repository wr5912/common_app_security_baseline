# 维护工作流

## 1. 新来源进入

```text
终端观测 / 官方文档 / EDR事件 / Sysmon事件 / 人工分析
  -> kb/raw_sources/ 或 kb/10_来源与证据/
```

## 2. AI 摘要

AI 先生成来源摘要，不直接改核心页面。

## 3. 映射实体

判断来源涉及：

```text
应用
服务
进程
父子关系
注册表
文件
网络
安全基线
```

## 4. 更新 Wiki

按模板更新对应页面。

## 5. 人工审核

重要安全判断必须审核：

```text
normality
risk_level
allowlist_condition
false_positive_condition
recommended_action
```

## 6. 变更记录

所有关键变更写入：

- [[变更日志]]
- [[待办事项]]
